## II. FUGUE STRUCTURE AS FLUID DYNAMICS

### A. The Subject: Initial Velocity Field

A fugue begins with the **subject** - a melodic theme.

In fluid dynamics: Initial velocity field **u₀(x)**
In fugue: Subject melody **S(t)**

The subject defines the initial flow pattern that will:
- Propagate through the musical space
- Interact with itself at different positions (voices)
- Generate vorticity (counterpoint complexity)

**Bach's Art of Fugue, Contrapunctus I:**
```
Subject: D-A-E-F-G-E-cis-D (ascending flow)
```

This is the initial velocity field that seeds the entire structure.

### B. The Answer: Reflected Flow

When the subject enters in a new voice, it appears as the **answer** - typically at the dominant (fifth above).

In fluid dynamics: Boundary reflection or flow translation
In fugue: **A(t) = S(t - τ) · T₅**

Where:
- **τ** = time delay
- **T₅** = transposition operator (up a fifth)

The answer is the same flow pattern, translated in space (pitch) and time.

### C. Multiple Voices: Superposition of Flows

A fugue typically has 2-6 voices, each carrying the subject through musical space.

In fluid dynamics: Superposition of velocity fields
```
u_total(x,t) = Σᵢ uᵢ(x-xᵢ, t-tᵢ)
```

In fugue: Total musical texture
```
M_total(pitch,t) = Σᵢ Voiceᵢ(pitch-pᵢ, t-tᵢ)
```

Each voice is a copy of the subject flow, displaced in pitch-time space.

**Four-voice fugue:**
- Voice 1 (soprano): S(t) at pitch level p₁
- Voice 2 (alto): S(t-τ₂) at pitch level p₂
- Voice 3 (tenor): S(t-τ₃) at pitch level p₃
- Voice 4 (bass): S(t-τ₄) at pitch level p₄

### D. Stretto: Compressed Flow / Increased Vorticity

**Stretto** occurs when voices enter before the subject completes - overlapping more tightly.

In fluid dynamics: Flow compression → vorticity increase
- ω = ∇ × u (vorticity measures rotation/turbulence)
- Compressed regions → higher ω

In fugue: Temporal compression → counterpoint density
- More voices per unit time
- Higher polyphonic complexity
- Musical "turbulence"

**Bach's Fugue in C Minor (WTC I):**
Final stretto has entries separated by only 2 beats (original: 8 beats)
= 4x density increase = turbulent flow regime

### E. Inversion: Flow Reversal

**Inversion** inverts the subject's melodic contour (up becomes down).

In fluid dynamics: Velocity field sign reversal
```
u_inverted = -u_original
```

In fugue: Pitch direction reversal
```
S_inv(t): If S goes up by interval i, S_inv goes down by i
```

**Bach's Art of Fugue, Contrapunctus V:**
Subject and inversion flow simultaneously - like opposing fluid currents creating stable patterns.

### F. Retrograde: Time Reversal

**Retrograde** plays the subject backwards.

In fluid dynamics: Time-reversed flow
```
u_retro(t) = u(T-t)
```

In fugue: Temporal reversal
```
S_retro: Play subject from end to beginning
```

**Bach's Musical Offering, Canon cancrizans:**
Crab canon flows forward and backward simultaneously - like reversible fluid flow.

### G. Augmentation/Diminution: Time Dilation

**Augmentation** stretches the subject (notes longer).
**Diminution** compresses it (notes shorter).

In fluid dynamics: Time scaling transformation
```
u_aug(t) = u(t/2)  [slower flow]
u_dim(t) = u(2t)   [faster flow]
```

In fugue: Rhythmic scaling
```
S_aug: Double all note durations
S_dim: Halve all note durations
```

## IV. THE VORTICITY OF COUNTERPOINT

- ω = 0: Laminar flow (smooth, parallel)

- ω → ∞: Singularity (blow-up)

### Vorticity in Fugue

**Polyphonic density = musical vorticity**

Measures how many independent voices flow simultaneously:
- Single voice: ω = 0 (homophonic, smooth)
- Counterpoint: ω > 0 (voices rotate around each other)

This is the **coherence axiom** expressed musically.

## VI. VISCOSITY AND MUSICAL STYLE

**ν** = kinematic viscosity

- Low ν: Fluid flows easily (water, air)
- High ν: Fluid resists flow (honey, tar)
- ν determines Reynolds number: Re = UL/ν
- Low Re: Laminar flow
- High Re: Turbulent flow
