## The Critical Transition Point

**Condition**: Verification time >> Iteration time

**Example**: Traditional peer review
- Iteration time: Days to weeks (write paper)
- Verification time: Weeks to months (peer review)
- **Bottleneck**: Waiting for verification

**Dynamics**:
- Sequential (one attempt, wait, revise)
- Conservative (avoid risky ideas - waste of time if wrong)
- Slow convergence (few attempts before deadline)
