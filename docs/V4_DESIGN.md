## Quality Standards for v4

### Every Statement Must Be:
- **Precise**: No hand-waving
- **Justified**: Proof, citation, or marked as conjecture
- **Connected**: Clear relationship to framework
- **Necessary**: Contributes to core argument

### Every Proof Must Have:
- Clear hypotheses
- Logical flow
- No circular dependencies (or explicitly marked)
- Reference to prior results

### Every Conjecture Must Have:
- Explicit label as "Conjecture"
- Justification for plausibility
- Testable consequences
- Or path to resolution
