# Empirical Operator Demos

Prereqs: Python 3.10+, numpy, scipy, matplotlib.

Install:
```
pip install -r requirements.txt
```

Run:
```
python3 experiments/nagarjuna_operators.py
```
Artifacts:
- Matrices, commutators, norms.
- Goal: [D̂, □] ≠ 0 and [D̂, □]^2 ≈ 0 (autopoietic nilpotency).

Optional:
```
python3 experiments/twelve_fold_test.py
python3 experiments/tower_growth_empirical.py
```
